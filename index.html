¿Cómo se ve?

El diseño es limpio, moderno y envolvente. El usuario verá un lienzo donde los gráficos reaccionan automáticamente a la música:

  • Colores que cambian según la emoción
  • Formas que se expanden con los bajos
  • Ondas suaves que siguen la melodía
  • Partículas que vibran al ritmo

¿Cómo funciona?

  1. Sube tu canción.
  2. El sistema analiza las frecuencias y ritmos (FFT).
  3. Se genera la visualización en tiempo real.
  4. Personaliza estilos, colores y patrones.

Instrucciones de Uso

Fuentes de Audio

  • Demos: Haz clic en "Audio de demostración" para probar 8 canciones incluidas (Metal, Rock, Pop, Techno, Reggae, Salsa, Instrumental, Nota pura).
  • Archivo local: Haz clic en "Cargar archivo" para subir tus propios MP3, WAV, OGG o M4A.

Controles del Visualizador

  • Animaciones: Haz clic en "Tipo de animación" para elegir entre Onda, Barras, Estrellas, Puntos o Curvas.
  • Paletas de colores: Haz clic en "Paleta de colores" para elegir entre Neón, Cálida o Fría. Los colores cambian según las frecuencias del audio (bajos → rojos/oscuros, agudos → azules/claros).
  • Historial: Haz clic en el ícono de menú en la esquina superior izquierda para ver las últimas canciones reproducidas.
  • Pantalla completa: Haz clic en el ícono de expandir para maximizar la visualización.
  • Configuración: Haz clic en el engranaje para acceder a más opciones de reproducción paletas de colores.

Nota importante sobre la carga

El sitio está alojado en un servidor gratuito que se "duerme" tras 15 minutos de inactividad. La primera visita puede tardar entre 30 y 50 segundos mientras el servidor se reactiva. Las siguientes cargas serán instantáneas (1-3 segundos).

¿Qué es el Visualizador Musical Web?

El Visualizador Musical Web es una aplicación interactiva que convierte señales de audio en representaciones gráficas dinámicas y sincronizadas. Utilizando tecnologías web modernas y análisis espectral avanzado (FFT), este proyecto permite visualizar música en tiempo real a través de múltiples modos: ondas, barras de frecuencia, partículas y estrellas.

Características Principales

  • Análisis FFT: Transformada Rápida de Fourier para descomponer audio en frecuencias.
  • 4 Visualizadores: Ondas, barras espectrales, partículas y estrellas sincronizadas.
  • Múltiples Fuentes: Carga archivos locales (MP3, WAV, OGG, M4A) o con demos.
  • Colores Dinámicos: Mapeo de frecuencias a colores neón.
  • Tiempo Real: Renderizado a 60 FPS con requestAnimationFrame.
  • Gratuito: Funciona directamente en el navegador.

Tecnologías aplicadas y conceptos

  • HTML5, CSS3, JavaScript ES6+
  • Web Audio API, Canvas API
  • FFT, análisis espectral por bandas, beat detection
  • requestAnimationFrame, teoría del color (HSL)
